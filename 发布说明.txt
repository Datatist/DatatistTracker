发布方法1：
    1， 克隆代码 https://gitlab.datatist.cn/datatist/datatist-analyzer-ios-tracker.git 
    2， 打开 DatatistTracker.xcodeproj，修改 DatatistTracker.m 中修改版本号，
        选中 DatatistTracker Scheme，编译
    3，选中 DatatistTracker_universal Scheme, 编译
    4，右键 libDatatistTracker.a, show in Finder
    5, 将 Release-universal 目录下的 libDatatistTracker-Release.a 复制到 lib 目录下，并重命名为 libDatatistTracker.a。
       将 include 目录下 .h 文件 复制到 lib 目录下。
修改DatatistTracker.podspec中的s.version为正确的版本号 2.0.0

    6, commit 并 push，用 SDK 的version，打 tag。例如： 如果 SDK 的版本号是 2.0.0，则 tag 就是 2.0.0   并上传tig. git push --tags
    7, pod spec lint DatatistTracker.podspec --allow-warnings --verbose验证一下.
    8，在 terminal 中执行 pod trunk register xxx@datatist.com 'xxx' --description='mackbook pro'
       （xxx@datatist.com 为公司邮箱， xxx 为邮箱用户名），执行成功后，会收到一封邮件。打开邮件，点击里面的连接。 pod trunk register jim.zhang@datatist.com 'xxx' --description='mackbook pro'
    9，cd 到 DatatistTracker cocoapods 库根目录，执行
       pod trunk push DatatistTracker.podspec --allow-warnings --verbose

    10, pod search 'DatatistTracker' 查看有没有最新的版本(会有一会的延时)
